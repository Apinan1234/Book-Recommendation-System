# 5️⃣ ตัวอย่าง PySpark Code (ใช้ใน Project)

## โหลดข้อมูล

```python
from pyspark.sql import SparkSession

spark = SparkSession.builder.appName("BookRecommendation").getOrCreate()

ratings = spark.read.csv("ratings.csv", header=True, inferSchema=True)
books = spark.read.csv("books.csv", header=True, inferSchema=True)
```

---

## รวมข้อมูล

```python
df = ratings.join(books, "book_id")
```

---

## หา Top Books

```python
from pyspark.sql.functions import avg, count

top_books = df.groupBy("title") \
    .agg(avg("rating").alias("avg_rating"),
         count("rating").alias("rating_count")) \
    .orderBy("avg_rating", ascending=False)

top_books.show(10)
```

---

# 6️⃣ ทำ Collaborative Filtering ด้วย ALS

Spark มี algorithm ชื่อ

```
ALS (Alternating Least Squares)
```

ใช้สำหรับ recommender system

```python
from pyspark.ml.recommendation import ALS

als = ALS(
    userCol="user_id",
    itemCol="book_id",
    ratingCol="rating",
    coldStartStrategy="drop"
)

model = als.fit(ratings)
```

---

## แนะนำหนังสือให้ user

```python
user_recommendations = model.recommendForAllUsers(5)

user_recommendations.show()
```
